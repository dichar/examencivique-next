# FAQ Produit (texte utilisateur)

## Combien de quiz gratuits ai-je ?
Vous disposez de 2 quiz gratuits pour tester la plateforme. Une fois ces 2 quiz terminés, l'accès aux nouveaux quiz passe par le Pack Réussite.

## Que comprend le Pack Réussite ?
Le Pack Réussite donne un accès complet à tous les quiz pendant 3 mois. Prix : 19,99 €.

## Le renouvellement est-il automatique ?
Non. Le pack expire après 3 mois et il faut repayer pour prolonger l'accès.

## Quand le paywall s'affiche-t-il ?
Après vos 2 quiz gratuits, le paywall s'affiche dès que vous tentez de démarrer un nouveau quiz.

## Que se passe-t-il si je ne suis pas connecté ?
Vous pouvez lancer un quiz. À la fin, un bouton "Voir mes résultats" vous invite à vous connecter. Après connexion, vous êtes redirigé vers vos résultats.

## Comment savoir si mon pack est actif ?
Dans votre compte, la date de fin d'accès est affichée (ex. "Actif jusqu'au 13/09/2026"). Tant que cette date est future, l'accès est illimité.

## Pas satisfait ?
Aucun remboursement n’est prévu. Deux quiz gratuits permettent d’essayer le service.
